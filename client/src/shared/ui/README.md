# shared/ui

Reusable presentational components. No business logic.

- **Button** — primary/secondary/danger variants, `isLoading` spinner
- **Input** — styled input with error border state
- **Label** — styled `<label>` with `.label` utility
- **FormField** — composes Label + Input + error message
- **Card** — glass container with border, shadow, blur
- **ErrorBanner** — inline error box for form submit errors
