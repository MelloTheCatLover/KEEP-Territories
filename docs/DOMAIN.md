## Базовые термины

- **сезон** — период времени, в течение которого проводится КТП. У каждого сезона
  свой порядковый номер, отсчёт ведётся с 56 смены (124 смена — 21 сезон).
- **съезд** — общее собрание всех участников для обсуждения и принятия законов.
- **борьба / война** — собрание всех команд, на котором захватываются сектора.
- **закон** — общеобязательное для всех команд положение, принимаемое на съездах
  голосованием. Может кардинально менять правила игры.
- **опыт** — условные единицы за захват секторов, повышают уровень команды.
- **уровень** — степень развития команды, растёт от количества опыта. За переход
  на следующий уровень команда получает одно очко апгрейда.
- **влияние** — условные единицы за захват секторов, определяют количество
  жетонов на съезде.
- **случайные встречи** — задания, за которые команды получают характеристики.
- **стрик** — непрерывная полоса занятий секторов.
- **перезахват** — команда берёт под контроль сектор, уже захваченный другой
  командой.
- **укрепление** — перезахват командой своего же сектора. Даёт половину влияния и
  опыта (в зависимости от сложности сектора).

## Характеристики

- **характеристики** — свойства, меняющие качества команды/персонажа через
  распределение очков апгрейда.
- **очки апгрейда** — единицы за новый уровень, команды распределяют их исходя из
  своей стратегии.

### Лидерство

Команда с наибольшим лидерством обладает правом вето.

**Право вето** — возможность отменить любой закон, даже принятый. Исключение —
законы, принимаемые председателем.

### Сила

Позволяет разрушать сектора с разным уровнем укрепления.

**Пробитие** — захват чужого укреплённого сектора с первой попытки.

| Очки силы | Пробитие |
|---|---|
| 1–4 | 0 |
| 5–7 | 1 |
| 8–9 | 2 |
| 10 | 3 |

### Выносливость

Позволяет перемещаться по захваченному полю на несколько секторов за один шаг.

**Очки передвижения** — возможность «перепрыгивать» по своим захваченным секторам.

| Очки выносливости | Очки передвижения |
|---|---|
| 1–3 | 0 |
| 4–6 | 1 |
| 7–9 | 2 |
| 10 | 3 |

### Интеллект

Позволяет совершать проверку сектора перед захватом.

**Проверка** — возможность увидеть задание сектора, не принимая его сразу.

| Очки интеллекта | Проверки |
|---|---|
| 1–2 | 0 |
| 3–4 | 1 |
| 5–6 | 2 |
| 7–8 | 3 |
| 9–10 | 4 |

### Удача

Позволяет совершать реролл при захвате сектора.

**Реролл** — возможность поменять задание на секторе.

| Очки удачи | Рероллы |
|---|---|
| 1–4 | 0 |
| 5–7 | 1 |
| 8–9 | 2 |
| 10 | 3 |

## Персонажи на карте

**Персонажи на карте** — персонажи на неизвестных участникам секторах,
позволяющие приобрести услугу или товар в помощь команде.

### Мастер

Товары, меняющие механику игры.

- **Раздвоение** — начать захват сразу двух секторов.
- **К.И.П** — копируй, инициализируй, побеждай.
- **Чип** — копирует один имплант у другой команды.
- **Высокий старт** — следующее особое событие начинается с преимуществом.
  Одноразовый.
- **Щит** — защита от одной диверсии.
- **LEVEL UP** — повышение уровня + бонус эры.

### Имплантер

Товары, улучшающие свойства команды, взамен забирают что-то нужное команде
(опыт, влияние и т. п.).

### Диверсант

Товары, тормозящие процесс игры.

- Очко покупки у следующего торговца.
- Снятие укреплений с одного сектора на выбор.
- Следующий сектор выполняется как сложный, команда отправляется на начальную
  позицию.
- Следующая разведка покажет ложь.
- Забрать пять влияния у соперника.
- Следующий ход вы совершаете за соперника.
- Соперник не получает награду за сектор.

### Торговец

Товары, изменяющие ход игры.

- **Батут** — два раза перепрыгнуть сектор.
- **Подзорная труба** — три раза разведать сектор.
- **Подушка безопасности** — дроп без последствий (стрик сбивается).
- **Кирпичи** — бесплатно укрепить один сектор.
- **Дополнительная рука** — плюс слот под один имплант.
- **Переборка** — поменять расстановку характеристик.

## Кубки

| Кубок | За что |
|---|---|
| Влиятельные | наибольшее влияние |
| Опытные | наибольший опыт |
| Универсальные | наибольшая сумма характеристик |
| Захватчики | наибольшее число перезахватов |
| Хранители ядра | захват ядра |
| Правители | наибольшее число секторов |
| Несгибаемые | наибольший стрик |
| Чемпионы | наибольшее число выигранных особых событий |

**Условие победы** — собрать наибольшее количество кубков.

## Правила КТП

**КТП** — массовая стратегия по захвату секторов в реальном времени.

**Основная цель** — получить наибольшее количество кубков, используя выбранную
стратегию. В реальном времени команда в своём темпе захватывает сектора разной
сложности, отталкиваясь от своей стратегии.

---

## Стратегии

Основные стратегии делятся на две группы.

**Качественная** — захват наиболее ценных и значимых секторов, даже если их
общее количество невелико. Акцент не на масштабе территории, а на её ценности:
максимальная выгода при меньших затратах ресурсов.

**Количественная** — захват большего числа секторов, каждый из которых
сравнительно менее ценен. Приоритет — расширение контролируемой территории, даже
если значимость отдельного сектора ниже.

### Отступления от текста

- **Передвижение.** Досягаемость считается от **последнего захваченного сектора**
  (якорь = дом на старте), а не от всей границы. Радиус = 1 + очки передвижения;
  без выносливости к далёкому сектору идут «перевалами» (промежуточными захватами).
- **Право вето** привязано к **влиянию** (команда с наибольшим влиянием), а не к
  лидерству — так решено при реализации. Одно вето на съезд.
- **Имплантер не реализован** сознательно. В `sectors.merchant_type` только
  `master` / `saboteur` / `trader`.
- Товары персонажей отыгрываются офлайн: сектор с персонажем выдаёт «жетон
  покупки», админ гасит его на `/admin/merchants` после визита команды.

### Расхождения в исходном тексте

- В определении **проверки** сказано «в зависимости от количества очков силы» —
  по смыслу и таблице это **интеллект**.
- Кубок «несгибаемые» описан через «стоик» — имеется в виду **стрик**.
